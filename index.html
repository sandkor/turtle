What is it?

Type commands in the notepad on the left side and press Enter.
Turtle will follow your instructions.

List of commands

fd X - move forward by X pixels
bd X - move backwards by X pixels
lt X - turn left by X degrees
rt X - turn right by X degrees
dir X - set turtle in a specific direction by X being degrees
ct - go to the center
goxy X Y - go to X and Y coordinates
gox X - go to X coordinate
goy Y - go to Y coordinate
pu - set pen state to up (off)
pd - set pen state to down (off)
pw X - Set pen width to X
pc R G B - Set pen color to R G B (0-255)
